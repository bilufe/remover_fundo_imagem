# remover_fundo_imagem

Esta é uma aplicação que tem o simples objetivo de remover o fundo de fotos para a preparação de uso em documentos.

## Incrementos realizados em 28 de dezembro de 2025:
- Adicionado código para verificar se a imagem tem um tamanho mínimo adequado para documentos;
- Adicionado código para verificar se a imagem tem um fundo branco;
- Adicionado código para verificar se a imagem é de rosto, ou se outras partes do corpo são exibidas;
- Adicionado código para verificar se a imagem contém apenas um rosto.

A aplicação utiliza a biblioteca Mediapipe do Google.
